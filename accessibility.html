Accessibility Statement

Last Updated: November 7, 2025

Our Commitment to Accessibility

HelpStash is committed to ensuring our iOS app is accessible to everyone, including people with disabilities. We follow the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA and Apple's Human Interface Guidelines for accessibility.

Our Goal: Make HelpStash usable by the widest possible audience, regardless of ability or technology.

Accessibility Features

VoiceOver Support

HelpStash fully supports Apple's VoiceOver screen reader:

  • ✓ All interactive elements have descriptive labels
  • ✓ Logical reading order on every screen
  • ✓ Timer countdown announces time remaining
  • ✓ Custom actions for common tasks (start timer, mark complete)
  • ✓ Notifications are read aloud when received

Tested on: iPhone 17 Pro with VoiceOver enabled + Screen Curtain

Dynamic Type

Text scales automatically with system text size preferences:

  • ✓ Supports all 12 system text sizes (Extra Small to AX5)
  • ✓ Layouts adapt to larger text without truncation
  • ✓ Timer countdown uses tabular figures (no layout shift)
  • ✓ Tested at 200%+ scaling (AX5)

How to adjust: Settings → Display & Brightness → Text Size

Reduce Motion

Simplified animations for users with vestibular disorders:

  • ✓ Breathing exercise circle uses opacity changes (not scale animation)
  • ✓ Timer transitions use cross-fade (not slide)
  • ✓ Number tile selection uses opacity (not pulse animation)
  • ✓ Haptic feedback remains enabled (separate from visual motion)

How to enable: Settings → Accessibility → Motion → Reduce Motion

Voice Control

All actions accessible via voice commands:

  • ✓ "Tap Start Pause" - Begins timer creation
  • ✓ "Tap Privacy" - Opens Privacy Policy
  • ✓ Custom voice labels for ambiguous elements

How to enable: Settings → Accessibility → Voice Control

High Contrast Mode

Enhanced color contrast for low vision users:

  • ✓ Text meets WCAG AA minimum (4.5:1 ratio)
  • ✓ Interactive elements clearly distinguishable
  • ✓ Pause allowance indicators use ADA-compliant colors (≥4.5:1)
  • ✓ Border contrast increases in Increase Contrast mode (8.6:1)

How to enable: Settings → Accessibility → Display & Text Size → Increase Contrast

Color Blind Support

Accessible to users with color vision deficiency:

  • ✓ 2:1 lightness ratio between filled/empty states
  • ✓ Tested with Deuteranopia, Protanopia, Tritanopia filters
  • ✓ Icons and text provide additional context (not color-only)

Test with: Settings → Accessibility → Display & Text Size → Color Filters

Touch Accommodations

Designed for users with motor impairments:

  • ✓ Touch targets ≥44×44 points (Apple HIG minimum)
  • ✓ Primary actions use ≥48×48 points (HelpStash standard)
  • ✓ Generous spacing between interactive elements
  • ✓ No time-sensitive interactions (no auto-dismiss)

Testing & Validation

Automated Testing

  • XCUITest accessibility suite (iOS 18+ Accessibility APIs)
  • Xcode Accessibility Inspector
  • Color contrast validation (Xcode + WebAIM)

Manual Testing

  • Reduce Motion enabled
  • Voice Control for primary tasks

Devices Tested

  • iPhone SE (3rd gen) - Smallest display
  • iPhone 17 Pro - Standard display
  • iPhone 17 Pro Max - Largest display

User Guides

VoiceOver Navigation Guide

  1. Launch app: Swipe right to hear "HelpStash. Button. Pause a Spend."
  2. Start timer: Double-tap on "Pause a Spend" button
  3. Enter amount: Swipe to amount field → Double-tap → Type with VoiceOver keyboard
  4. Select duration: Swipe to duration picker → Swipe up/down to adjust
  5. Confirm: Swipe to "Start Pause" → Double-tap

Dynamic Type Setup

  1. Settings → Display & Brightness → Text Size
  2. Drag slider to preferred size
  3. For larger sizes: Settings → Accessibility → Display & Text Size → Larger Text
  4. Enable "Larger Accessibility Sizes" for AX1-AX5
